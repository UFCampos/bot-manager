## Requisitos previos:

Asegúrese de tener Node.js instalado en su sistema con una versión igual o superior a la 18.17.1.


## Instrucciones para iniciar el bot:

Abra una consola de comandos (Powershell o GitBash) en la carpeta raíz del bot. Para hacerlo, puede presionar Shift + clic derecho en la carpeta y seleccionar "Abrir ventana de PowerShell".

Instale las dependencias necesarias ejecutando el siguiente comando:
npm install

Una vez que las dependencias se hayan instalado correctamente, puede iniciar el bot con el siguiente comando:
npm start

El bot mostrará un código QR en la consola. Utilice la aplicación WhatsApp en su dispositivo móvil para escanear este código QR.

Tenga en cuenta que la carga y confirmación de la conexión pueden llevar un tiempo. Si no ve el código QR o experimenta problemas de conexión, siga estos pasos:

Para reiniciar la aplicación, presione Ctrl + C en la consola para detener el bot.
Luego, vuelva a ejecutar el bot con el comando npm start.
No es necesario volver a escanear el código QR si está reejecutando la aplicación; simplemente espere a que se establezca la conexión.
Si en algún momento cierra la sesión y no puede ver el código QR, repita el proceso anterior para reiniciar la aplicación y escanear el código QR nuevamente.
